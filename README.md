# Cellfinder-explore User Guide

## Installation


```conda create -n cellfinder_explore python=3.8```

```conda activate cellfinder_explore```

```git clone https://github.com/stephenlenzi/cellfinder-explore```

```cd cellfinder-explore```

```pip install -e .```
